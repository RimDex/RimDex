# Development Guide

RimDex is open source under a copy-left license. We welcome all forms of contributions whenever it is making bug reports via [issues on GitHub][Issues] or code contributions via pull requests!

The source code is fully available and RimDex can be run using a Python interpreter or self-compiled.

Contributions should follow [our Code of Conduct](https://github.com/RimDex/RimDex/tree/main/CODE_OF_CONDUCT.md) and [Contributor Guidelines](../development-guide/contributor-guidelines).

[Issues]: https://github.com/RimDex/RimDex/issues
